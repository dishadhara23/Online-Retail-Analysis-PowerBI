## Overview
End-to-end data analysis project using a real UK online retail dataset 
(500,000+ transactions, Nov 2010 – Dec 2011).

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI Desktop

## Project Steps
1. Data Cleaning — removed cancelled orders, handled missing records
2. EDA — revenue trends, top products, seasonal demand
3. Business Analytics — customer lifetime value, Pareto analysis
4. Data Visualisation — multi-page Power BI dashboard with DAX measures

## Key Findings
- Top 20% of customers generate 80% of revenue (Pareto rule confirmed)
- Revenue spikes sharply in Q4 (Oct-Nov) every year
- UK dominates revenue but Netherlands and EIRE are strong markets

## Dataset
Online Retail Dataset by Ulrik Thyge Pedersen — Kaggle
